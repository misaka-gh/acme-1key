# Acme.sh 域名证书一键申请脚本

此脚本可以帮助你使用acme.sh脚本申请域名的ssl证书

## 使用方法

```shell
wget -N https://raw.githubusercontent.com/Misaka-blog/acme-1key/master/acme1key.sh && bash acme1key.sh
```

快捷方式 `bash acme1key.sh`

说明文档：https://owo.misaka.rest/acme-1key/
