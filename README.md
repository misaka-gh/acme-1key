# Acme.sh 域名证书一键申请脚本

此脚本可以帮助你使用acme.sh脚本申请域名的ssl证书，并且可以保存到你想要的位置

## 使用方法

```shell
wget -N https://cdn.jsdelivr.net/gh/Misaka-blog/acme1key@master/acme1key.sh && chmod -R 777 acme1key.sh && bash acme1key.sh
```

快捷方式 `bash acme1key.sh`
